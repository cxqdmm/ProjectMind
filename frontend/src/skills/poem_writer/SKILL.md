# 用途
用于根据给定主题与体裁创作中文诗歌，并提供自检与润色建议，产出可直接展示与复用的成品。

## 生成所需信息
- 主题：诗歌的核心意象或场景（必填）
- 体裁：现代诗 / 五言绝句 / 七言绝句 / 词（选填，默认七言绝句）
- 风格与语气：清新/凝练/豪放/婉约（选填）
- 约束：韵脚、是否藏头、长度（行数）、偏好意象（选填）
- 说明要求：是否需要在结果后附简要说明（体裁、韵脚、意境要点等）

## 参考与写作要点
以下参考文件提供不同体裁的写作要点与范式：
- 现代诗：`references/modern.md`
  - 要点：意象连贯、自由换行、节奏与重复、隐喻与留白
- 五言绝句：`references/wujue.md`
  - 要点：四句五字、起承转合、炼字凝练、意象集中
- 七言绝句：`references/qijue.md`
  - 要点：四句七字、节奏稳健、押韵规范、尾句有力
- 词：`references/ci.md`
  - 要点：词牌规范、平仄与节奏、意境铺陈、收尾含蓄

## 建议流程
1. 澄清并补全要素（主题、体裁、约束与风格）
2. 加载相应体裁的参考要点与必要的范式资料
3. 依据体裁与约束进行构思与创作，保持意象一致与节奏稳定
4. 按自检清单检查：韵脚统一、避免现代词、（律诗）对仗、情感连贯
5. 输出结构化结果，并在需要时附上简要说明（体裁、韵脚、意境要点）

## 输出格式
```
作品：
<诗文本，逐句分行>

说明：
- 体裁：<七绝/五律/现代诗/词...>
- 韵脚：<韵部或字样>
- 藏头：<若有>
- 意境要点：<简述>
```

## 自检与异常
- 信息不足：先澄清主题或体裁后再生成
- 生成失败：提示更明确的约束（目标意象与韵脚字样）
- 质量不佳：参考写作要点与范例，强化意象与节奏
